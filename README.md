# 🎮 TeammateFinder Bot

VK-бот для поиска тиммейтов в онлайн-игры. Механика в стиле «Дайвинчик» — свайпай анкеты, лайкай, находи матчи!

## Возможности

- **Анкеты** — ник, возраст, город, игры + ранг + роль, микрофон, фото
- **Поиск** — просмотр анкет с кнопками 👍/👎/💬
- **Матчи** — уведомления при взаимном лайке
- **Фильтры** — по игре, рангу, городу, возрасту, микрофону
- **Избранное, жалобы, блокировки**
- **Рейтинг** игроков
- **Админ-панель** — статистика, модерация, бан/разбан
- **Антиспам** — лимит 30 лайков/день (Premium — безлимит)

### Поддерживаемые игры
CS2, Dota 2, Valorant, Apex Legends, League of Legends, Overwatch 2, PUBG, Fortnite

## Быстрый старт

### 1. Настройка VK

1. Создай сообщество VK → Управление → Работа с API
2. Создай ключ доступа (токен) с правами на сообщения
3. Включи Long Poll API v5.199, события: `message_new`, `message_event`
4. Включи возможности ботов → «Разрешить отправку сообщений»

### 2. Запуск

```bash
# Клонируй и перейди в папку
cd teammate-bot

# Установи зависимости
pip install -r requirements.txt

# Настрой переменные
cp .env.example .env
# Отредактируй .env — укажи токен, ID группы, ID админов

# Запуск
python bot.py
```

### 3. Переменные окружения

| Переменная | Описание |
|---|---|
| `VK_TOKEN` | Токен сообщества VK |
| `VK_GROUP_ID` | ID группы VK |
| `ADMIN_IDS` | VK ID админов через запятую |
| `DB_PATH` | Путь к SQLite базе (по умолчанию `teammate_bot.db`) |

## Структура проекта

```

├── bot.py          # Основной файл бота (LongPoll + handlers)
├── config.py       # Конфигурация, список игр/рангов/ролей
├── database.py     # SQLite: пользователи, лайки, матчи, фильтры
├── keyboards.py    # VK-клавиатуры
├── utils/
│   └── __init__.py # Форматирование профилей

```

## Админ-команды

В чате с ботом (только для ID из `ADMIN_IDS`):
- `/admin` — панель
- `/ban <vk_id>` — бан
- `/unban <vk_id>` — разбан
- `/resolve <report_id>` — закрыть жалобу

## Расширение

- Добавить игры → `config.py` → словарь `GAMES`
- Миграция на PostgreSQL → заменить `database.py` на asyncpg
- Telegram-версия → заменить VK LongPoll на aiogram
- Discord интеграция → webhook уведомления
